---
title: "INSTALL_003 - Invalid Install Method"
description: "The content type can't be installed using the requested method."
---

## What This Means

The content type you're trying to install isn't supported by the target provider. Not all content types are compatible with every provider.

## Common Causes

- Trying to install a content type the target provider doesn't support (e.g., hooks to a provider without a hook system)
- Mismatch between the content type and the provider's capabilities

## How to Fix

1. Check provider compatibility with `syllago info providers` to see which content types each provider supports.
2. Review the [format conversion](/syllago-docs/using-syllago/format-conversion/) page for the full compatibility matrix.
3. Consider using a different target provider that supports the content type.

## Example Output

```
Error [INSTALL_003]: Invalid install method
Suggestion: Check provider compatibility with 'syllago info providers'.
Docs: https://openscribbler.github.io/syllago-docs/errors/install-003/
```
