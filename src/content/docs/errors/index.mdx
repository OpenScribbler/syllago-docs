title: Error Codes
description: Reference for all syllago error codes with explanations and fixes.

When syllago encounters an error, it displays a structured error with a code, message, and suggestion. Each code links to its documentation page below.

## Catalog Errors

| Code | Description |
|------|-------------|
| [CATALOG_001](/syllago-docs/errors/catalog-001/) | No catalog found |
| [CATALOG_002](/syllago-docs/errors/catalog-002/) | Catalog scan failed |
