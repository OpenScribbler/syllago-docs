---
title: Error Codes
description: Reference for all syllago error codes with explanations and fixes.
---

When syllago encounters an error, it displays a structured error with a code, message, and suggestion. Run `syllago explain <CODE>` for offline help, or click any code below for the full reference.

## Catalog Errors

| Code | Name | Description |
|------|------|-------------|
| [CATALOG_001](/errors/catalog-001/) | Catalog Not Found | no syllago repo or library found |
| [CATALOG_002](/errors/catalog-002/) | Catalog Scan Failed | catalog scan returned an error |

| [CONFIG_002](/errors/config-002/) | Config Not Found | configuration file does not exist |
| [CONFIG_003](/errors/config-003/) | Config Path | invalid path override (not absolute, unknown type) |
| [CONFIG_004](/errors/config-004/) | Config Save | failed to save configuration |
