<Aside type="caution" title="Deprecated">
`syllago promote to-registry` has been replaced by [`syllago publish`](/syllago-docs/using-syllago/cli-reference/publish/).
</Aside>

```sh
# Old (deprecated)
syllago promote to-registry my-registry skills/my-skill

# New
syllago publish my-skill --registry my-registry
```

See [`syllago publish`](/syllago-docs/using-syllago/cli-reference/publish/) for full documentation.
