---
title: "syllago telemetry"
description: "Manage usage analytics settings"
---

View and control anonymous usage data collection. Run 'syllago telemetry status' for details.

## Subcommands

| Command | Description |
|---------|-------------|
| [`off`](/using-syllago/cli-reference/telemetry-off/) | Disable telemetry |
| [`on`](/using-syllago/cli-reference/telemetry-on/) | Enable telemetry |
| [`reset`](/using-syllago/cli-reference/telemetry-reset/) | Generate a new anonymous ID |
| [`status`](/using-syllago/cli-reference/telemetry-status/) | Show telemetry state and anonymous ID |
