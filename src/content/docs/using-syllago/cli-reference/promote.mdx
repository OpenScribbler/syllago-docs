import { Aside } from '@astrojs/starlight/components';

<Aside type="caution" title="Deprecated">
`syllago promote` has been replaced by two commands:
- [`syllago share`](/syllago-docs/using-syllago/cli-reference/share/) — Contribute to a team repo
- [`syllago publish`](/syllago-docs/using-syllago/cli-reference/publish/) — Contribute to a registry
</Aside>

The `promote` command has been split:

```sh
# Old (deprecated)
syllago promote to-registry my-registry skills/my-skill

# New — share with your team
syllago share my-skill

# New — publish to a registry
syllago publish my-skill --registry my-registry
```

See [`syllago share`](/syllago-docs/using-syllago/cli-reference/share/) and [`syllago publish`](/syllago-docs/using-syllago/cli-reference/publish/) for full documentation.
