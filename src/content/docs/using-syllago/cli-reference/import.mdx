<Aside type="caution" title="Deprecated">
`syllago import` has been renamed to [`syllago add`](/syllago-docs/using-syllago/cli-reference/add/).
</Aside>

The `import` command has been replaced by `add`. Use:

```sh
# Old (deprecated)
syllago import --from claude-code

# New
syllago add --from claude-code
```

See [`syllago add`](/syllago-docs/using-syllago/cli-reference/add/) for full documentation.
