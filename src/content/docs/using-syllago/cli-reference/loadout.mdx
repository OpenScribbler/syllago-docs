---
title: "syllago loadout"
description: "Apply, create, and manage loadouts"
---

Loadouts bundle a curated set of syllago content — rules, hooks, skills,
agents, MCP servers — into a single shareable configuration.

Use "syllago loadout apply" to try or apply a loadout.
Use "syllago loadout create" to build a new loadout interactively.
Use "syllago loadout remove" to revert an active loadout.

## Subcommands

| Command | Description |
|---------|-------------|
| [`apply`](/using-syllago/cli-reference/loadout-apply/) | Preview or apply a loadout (default: preview; use --try or --keep to apply) |
| [`create`](/using-syllago/cli-reference/loadout-create/) | Interactively create a new loadout |
| [`list`](/using-syllago/cli-reference/loadout-list/) | List available loadouts |
| [`remove`](/using-syllago/cli-reference/loadout-remove/) | Remove the active loadout and restore original configuration |
| [`status`](/using-syllago/cli-reference/loadout-status/) | Show active loadout status |
