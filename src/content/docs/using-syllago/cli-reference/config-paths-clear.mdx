---
title: "syllago config paths clear"
description: "Clear path overrides for a provider"
---

## Synopsis

```
syllago config paths clear <provider>
```

## Description

Removes all path overrides for a provider, reverting it to default locations.

Examples:
```sh
syllago config paths clear claude-code
```
