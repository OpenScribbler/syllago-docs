## Synopsis

```
syllago config paths set <provider> --base-dir <path>
```

## Description

Sets a custom base directory for a provider. This is useful when a provider
is installed at a non-default location.

Examples:
```sh
# Set a custom base directory for Claude Code
syllago config paths set claude-code --base-dir ~/custom/claude

# Set a custom base for Cursor
syllago config paths set cursor --base-dir /opt/cursor
```

## Options

| Flag | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| `--base-dir` | `string` | — | Yes | Custom base directory path |
