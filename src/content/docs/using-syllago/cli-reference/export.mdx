import { Aside } from '@astrojs/starlight/components';

<Aside type="caution" title="Deprecated">
`syllago export` has been replaced by two commands:
- [`syllago install`](/syllago-docs/using-syllago/cli-reference/install/) — Activate content in a provider
- [`syllago convert`](/syllago-docs/using-syllago/cli-reference/convert/) — Convert content for sharing
</Aside>

The `export` command has been split:

```sh
# Old (deprecated)
syllago export --to cursor

# New — install content into a provider's location
syllago install my-skill --to cursor

# New — convert for ad-hoc sharing (stdout)
syllago convert my-skill --to cursor
```

See [`syllago install`](/syllago-docs/using-syllago/cli-reference/install/) and [`syllago convert`](/syllago-docs/using-syllago/cli-reference/convert/) for full documentation.
