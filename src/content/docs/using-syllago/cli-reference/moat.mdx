title: "syllago moat"
description: "Manage MOAT registry trust and verification"
---

Inspect and manage the MOAT (Model for Origin Attestation and Trust) subsystem.

Slice 1 surface: trusted-root lifecycle via `moat trust status`.
Later slices will add manifest verification, tier policy, and revocation.

## Subcommands

| Command | Description |
|---------|-------------|
| [`sign`](/using-syllago/cli-reference/moat-sign/) | Produce a signature.bundle from a manifest (online or dev-offline mode) |
| [`trust`](/using-syllago/cli-reference/moat-trust/) | Trusted-root lifecycle (status, refresh) |
