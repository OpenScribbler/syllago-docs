## Synopsis

```
syllago config paths show [--provider <name>]
```

## Description

Shows all configured path overrides, or overrides for a specific provider.

Examples:
```sh
# Show all path overrides
syllago config paths show

# Show overrides for a specific provider
syllago config paths show --provider claude-code
```

## Options

| Flag | Type | Default | Required | Description |
|------|------|---------|----------|-------------|
| `--provider` | `string` | — | No | Show overrides for this provider only |
