    <dd class="provider-extensions__description">When Copilot delegates work to a custom agent, the agent runs as a subagent with its own isolated context window. This allows large or specialized tasks to be offloaded without polluting the main agent's context. The main agent handles high-level planning and coordination while subagents handle execution. Versioning is Git-SHA-based: the agent profile in use when a task is assigned remains the active version for the lifetime of that task (pull request interactions use the same version for consistency).</dd>
