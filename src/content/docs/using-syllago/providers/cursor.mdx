| Content Type | Location | Format |
|-------------|----------|--------|

## Format Notes

Cursor uses MDC (Markdown Component) format rather than plain Markdown. MDC files have a metadata header block delimited by `---` that contains fields like `description`, `globs`, and `alwaysApply`. Syllago automatically converts between standard Markdown and MDC format when adding from or installing to Cursor.

Example MDC structure:

```
---
description: Rule description here
globs: "**/*.ts"
alwaysApply: false
---

Rule content in Markdown.
```

## Working with Cursor

```bash
# Add content from Cursor
syllago add --from cursor

# Install content to Cursor
syllago install my-rule --to cursor
```
